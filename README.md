# Car Price Prediction Deployment

An end-to-end Machine Learning project that predicts the selling price of used cars using Random Forest Regression and Flask.

---

# Project Overview

This project predicts the resale price of a car based on features such as:
- Year
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission Type
- Number of Owners

The model is trained using Machine Learning and deployed using Flask.
